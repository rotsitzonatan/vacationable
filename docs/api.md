# API

## API Style
RESTful API

## Authentication
JWT-based authentication with role-based authorization.

## Main Resources
- /auth
- /users
- /hotels
- /rooms
- /bookings
- /reviews
- /images
- /amenities

## Response Format
The API will use JSON for both requests and responses.

## Versioning
The API will initially use the `/api/v1` prefix.
